## Timing guidance

- **Day 0**: confirm GitHub repo, push HF Space.
- **Day 1**: NVIDIA forum post, X thread, Show HN (Tues-Wed morning EST).
- **Day 2-3**: HF Blog submission, QPU vendor outreach emails.
- **Week 2-4**: track traction. Once you have 50+ GitHub stars or a trending HF Space, pitch NVIDIA Developer Blog and quantum trade pubs.
- **Week 4+**: NVIDIA Inception application, referencing the public artifacts and any traction.

## Rules that apply to everything

- No em dashes anywhere (universal Crowe Logic style rule).
- No emojis unless Crowe Logic has commissioned a custom one for the piece.
- Attribute NVIDIA, Hugging Face, and the Qiskit/PennyLane/Cirq ecosystem by name wherever they appear. Cross-posts are earned by honoring partners' work.
- Numbers beat adjectives. Iteration counts, convergence thresholds, model parameter counts, error rates.
